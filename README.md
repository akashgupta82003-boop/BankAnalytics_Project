1.Project Title / Headline
Bank Analytics Dashboard | Credit, Debit & Loan Performance Analysis
*Project OverView: The Bank Analytics Dashboard is a comprehensive data visualization project designed to analyze and monitor key banking metrics such as credit, debit, loan funding, and collection performance.
This project aims to provide clear insights into customer behavior, transaction trends, and loan performance to support data-driven decision-making in the banking sector.

2.Short Description / Purpose
*The main goal of this project is to:
*Analyze financial transaction patterns (credit & debit).
*Track loan performance and repayment trends across various states and age groups.
*Build an interactive Power BI dashboard that enables quick insight into key banking KPIs.
*Demonstrate end-to-end data analytics workflow from data cleaning, processing, and visualization
